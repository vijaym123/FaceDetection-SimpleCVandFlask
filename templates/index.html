About the project : 
  Face detection is a well known problem in the world of Computer 
Vision and Machine Learning. I have mainly used  SimpleCV(Computer
Vision library in Python) , Flask (Microframework for web development) 
and  Amazon web services for hosting my project. Face detection is
achieved using a Haar classifier. This is a type of wavelet classifier which 
is trained to detect faces fast and efficiently. It will iterate over the 
image to find faces on different scales.
